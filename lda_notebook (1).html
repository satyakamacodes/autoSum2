Problem statement

  • The bank note authentication dataset is obtained from the UCI Machine Learning Repository

  • The task is to classify a new note as REAL or FAKE using LDA.

  • The features in the dataset are:

Feature name Meaning Encoded as Type
variance variance of the wavelet transformed image variance continuous
skewness skewness of the wavelet transformed image skewness continuous
kurtosis kurtosis of the wavelet transformed image kurtosis continuous
entrophy entropy of the image entrophy continuous
class whether the note is Real or Fake class categorical

Mathematical formulation of LDA

Let us assume a \(m\) dimensional sample space \[\{x^{1}, x^{2}, .... , x^{N} \}\] where \(N_{1}\) samples belong to \(w_{1}\) class and \(N_{2}\) samples belong to \(w_{2}\), \(N = N_{1}+N_{2}\)

Let us also consider a projection \(y\) such that \(y = w^{T}x\) where \(x = \begin{bmatrix} x_{1} \\ x_{2} \\ \vdots\\ x_{m} \end{bmatrix}\) and \(w = \begin{bmatrix} w_{1} \\ w_{2} \\ \vdots\\ w_{m} \end{bmatrix}\) where \(w\) is the projection vector used for projecting \(x\) on \(y\)

Mean of x in each class \[\mu_{i} = \frac{1}{N} \sum_{x \epsilon w_{i}}x\] and mean of y in each class \[\tilde{\mu_{i}} = \frac{1}{N}\sum_{y \epsilon w_{i}}y = \frac{1}{N}\sum_{x \epsilon w_{i}}w^{T}x = w^{T}\frac{1}{N}\sum_{x\epsilon w_{i}}x = w^{T}\mu_{i}\]

i.e.projecting x on y \(\Rightarrow\) projecting the mean of x on the mean of y

Thus distance between projected means can be written as \[J(w) = |\tilde{\mu_{1}}-\tilde{\mu_{2}}| = |w^{T}\mu_{1} - w^{T}\mu_{2}| = |w^{T}(\mu_{1}-\mu_{2})| \] In each class, scatter or variability of a projection is \[\tilde{s_{i}^{2}} = \sum_{y \epsilon w_{i}}(y - \tilde{\mu_{i}})^{2}\] and the within class variability after projection is \(\tilde{s_{1}^{2}}+\tilde{s_{2}^{2}}\)

Thus the objective function is \[J(w) = \frac{|\tilde{\mu_{1}} - \tilde{\mu_{2}}|^{2}}{\tilde{s_{1}^{2}}+\tilde{s_{2}^{2}}}\] needs to be maximized wrto. \(w\) i.e. \(|\tilde{\mu_{1}} - \tilde{\mu_{2}}|^{2}\) needs to be \(\Uparrow\) and \(\tilde{s_{1}^{2}}+\tilde{s_{2}^{2}}\) needs to be \(\Downarrow\)

\[\tilde{s_{i}^{2}} = \sum_{y\epsilon w_{i}}(y - \tilde{\mu_{i}})^{2} = \sum_{x\epsilon w_{i}}(w^{T}x - w^{T}\mu_{i})^{2} = \sum_{x\epsilon w_{i}}w^{T}(x - \mu_{i})(x - \mu_{i})^{T}w = w^{T}\Bigg( \sum_{x \epsilon w_{i}}(x - \mu_{i})(x - \mu_{i})^{T}\Bigg)w = w^{T}S_{i}w\] therefore \(\tilde{s_{1}^{2}}+\tilde{s_{1}^{2}} = w^{T}S_{1}w+w^{T}S_{2}w = w^{T}\Bigg(S_{1}+S_{2}\Bigg)w = w^{T}S_{w}w\) where \(S_{w}\) is within class scatter matrix

Also, \[\Bigg(\tilde{\mu_{1}} - \tilde{\mu_{2}}\Bigg)^{2} = \Bigg(w^{T}\mu_{1} - w^{T}\mu_{2}\Bigg)^{2} = w^{T}(\mu_{1}-\mu_{2})(\mu_{1}-\mu_{2})^{T}w = w^{T}S_{B}w\] where \(S_{B}\) is between class scatter matrix

therefore \[ J(w) = \frac{|\tilde{\mu_{1} -\tilde{\mu_{2}}}|^{2}}{\tilde{s_{1}}^{2} + \tilde{s_{2}}^{2}} = \frac{w^{T}S_{B}w}{w^{T}S_{W}w}\]

\[\frac{d}{dw}J(w) = \frac{d}{dw}\Bigg( \frac{w^{T}S_{B}w}{w^{T}S_{w}w} \Bigg) = 0\] \[\Rightarrow (w^{T}S_{w}w)\frac{d}{dw}(w^{T}S_{B}w) - (w^{T}S_{B}w)\frac{d}{dw}(w^{T}S_{W}w) = 0\]
